# hermes-skills
Skills do agente Hermes — automações pessoais e marketing
# Skill: marketing_shekina

Agente de marketing do Sítio Shekina.
Gera roteiros de Reel, legendas, copies e descrições para Airbnb
a partir de fotos enviadas pelo Telegram.

## Comandos
- `/reel 15s` `/reel 30s` `/reel 60s` — roteiros de vídeo
- `/airbnb` — descrição para Airbnb/Booking
- `/legenda` — legenda para post
- `/stories` — sequência de stories
- `/anuncio` — copy para impulsionar
- `/responder` — responde dúvidas de hóspedes

## Instalação
Copie `prompt.md` como system prompt da skill no Hermes
e configure o trigger para os comandos acima.
